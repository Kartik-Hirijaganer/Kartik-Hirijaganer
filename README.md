Production systems at Premier Health Group (client-owned source — full case studies on my [portfolio](https://kartik-hirijaganer.github.io/portfolio/)):

- **Release 1.0 of a HIPAA-compliant SaaS platform** — led a **6-person** team; scoped & architected **15+ features** across web and mobile, shipped in **15 weeks with no missed milestones**.

Earlier: built FinTech pipelines processing **$20M+/day** with **<5-min** freshness (Biz2X), and cut order cancellations **4% → <1%** debugging across 50M+ logs (Capgemini).
